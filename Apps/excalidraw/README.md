# Excalidraw Full

Collaborative web-based drawing application with an infinite canvas.

## Features
- Real-time collaborative drawing
- Hand-drawn look and feel
- Infinite canvas
- Multiple shape and connector tools
- Export to multiple formats

## Container Info
- **Image**: ghcr.io/joshevans-weber/excalidraw:latest
- **Port**: 5080
- **URL**: http://localhost:5080
